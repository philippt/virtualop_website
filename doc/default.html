The virtualop is a framework for automating the setup, deployment and operations of web applications.


The basic idea is to help you get your web app online and keep it running:

To accomplish this, we use the virtualop to setup a number of infrastructure services around your web application:

The specific requirements of how your application should be integrated are specified as part of your project's code source, in files that we call service descriptors.

When your application is integrated into it's runtime environment, it is deployed automatically onto a dedicated production environment consisting of a two-node cluster:

All data from your application is backed up regularly and stored in a data repository:

This data repository can be used to provide test data for setting up copies of your production system for development and testing:
