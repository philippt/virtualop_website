Installation

Requirements


  • Linux
  • Ruby (1.9.3)

Packages

RPM for RedHat/CentOS/SLES


There are RPM packages available at http://packages.dev.virtualop.org/rpm.

On a current CentOS (6.5), you'll have to add two RPM repositories like this: 

sudo wget -O /etc/yum/repos.d/epel.repo http://mirror.dev.virtualop.org/pub/Mirrors/epel/6/x86_64/epel-release-6-8.noarch.rpm
sudo wget -O /etc/yum.repos.d/virtualop.repo http://packages.dev.virtualop.org/rpm/virtualop.repo
If this works, you can install the virtualop with: 
sudo yum install -y virtualop
Note: This will attempt to install RVM from rvm.io systemwide and then install ruby 1.9.3 into the context of a newly created user called 'marvin'.
