The virtualop is about making the setup and operations of web applications as easy, fast and predictable as possible.

It is an open source framework written in Ruby that simplifies automating processes with the help of very simple ruby scripts that are integrated into a common runtime environment with a web dashboard and an own shell.

The basic idea is to aim for a level of automation where you're able to setup a complete platform from scratch with one click while being able to fine-tune or replace every single building block of the process within minutes.
